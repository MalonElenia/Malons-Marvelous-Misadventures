Separate from [[Carry Weight]], which determines raw lifting ability, **Inventory** represents how much stuff you can carry on your person. This is represented by Inventory Slots, which most characters have 20 of by default.

Other modifiers, such as [[Size and Reach|Size]] can affect inventory, with each size stage up or down from Medium granting +4 or -4 slots. [[STR]] gives +4 per point.

Every item from the biggest sword to the tiniest potion has an inventory slot stat, representing how annoying it is to carry around. Items do not consume inventory slots when you have them equipped to your [[Equipment Slot|Equipment Slot]], including your hands in a pinch.

Common pieces of equipment consume the following number of inventory slots.

| Equipment                                     | Slots                        |
| --------------------------------------------- | ---------------------------- |
| Light/1h Weapons                              | 1                            |
| 2h weapons (including most ranged weapons)    | 2                            |
| Ammunition                                    | 2                            |
| Light Armor                                   | 1                            |
| Medium Armor                                  | 2                            |
| Heavy Armor                                   | 3                            |
| Buckler                                       | 1                            |
| Shields                                       | 2                            |
| Most accessories, boots, gloves, and headgear | 1                            |
| Potions, toxins, and other alchemical items   | 1                            |
| Staff components                              | 1 (or 2 if also a 2h weapon) |
|                                               |                              |
| 1 tiny creature                               | 1                            |
| 1 small creature                              | 2                            |
| 1 medium creature                             | ~5                           |
| 1 large creature                              | ~20                          |
| 1 huge creature                               | ~50                          |

When denoting items on your character sheet, you can do the following to visually help represent filled inventory slots.

| Slots | Item               |
| ----- | ------------------ |
| 1     | Spare greatsword   |
| 2     | *                  |
| 3     | Spare medium armor |
| 4     | *                  |
| 5     | Bola               |
| 6     | Bola               |
| 7     | Dog                |
| 8     | *                  |
| Etc.  | *                  |

If you prefer physical sheets, doing something similar like drawing a line down the numbered lines for each additional slot an item takes up works too.
