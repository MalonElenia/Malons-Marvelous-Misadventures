A creature's **Carry Weight** represents what it is capable of lifting and moving around with without much effort at a moment's notice, such as when lifting a fallen object off of someone or moving an obstacle. This is independent from [[Inventory]], and common items in the Inventory or worn in [[Equipment Slot|Equipment Slots]] should not have their weight considered.

The default value is 150, but some factors may change this.

A creature can lift and drag twice as much as their carry weight, but is staggered while doing so.


| Bonuses to Carry Weight  |                                                                                                                                 |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------- |
| [[STR\|Strength]]        | +50% for each point, additive (or half for a negative point)                                                                    |
| [[Size and Reach\|Size]] | Quadruple Carry Weight (and weight) for each size above medium. (or 1/2 Carry Weight and 1/4 weight for each size below medium) |
